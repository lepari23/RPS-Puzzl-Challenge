# RPS-Puzzl-Challenge

Solution to Puzzl Intern Challenge to create Rock Paper Scissors game

## General ##

### Installation ###
First, create a virtual environment:
~~~~
python3 -m venv venv
~~~~

Activate virtual environment:
~~~~
. ./venv/bin/activate
~~~~

Next, install all requirements:
~~~~
pip install -r requirements.txt
~~~~

### Execution ###

~~~~
python3 game.py
~~~~

### Dependencies ###
* Pygame
* Numpy
* Tkinter
